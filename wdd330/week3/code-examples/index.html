Array.prototype.sort()

W3Schools

I never knew that the Array.prototype.sort() method allowed a function input... I thought it just sorted the array, though I guess it makes sense, because arrays sometimes have objects in them, and the regular sorting algorithm, can't sort objects in the correct way.

JavaScript
const someArray = [
    { name: "Someone1", rank: 56 },
    { name: "Someone2", rank: 25 },
    { name: "Someone3", rank: 73 },
    { name: "Someone4", rank: 15 },
    { name: "Someone5", rank: 5 },
    { name: "Someone6", rank: 26 },
    { name: "Someone7", rank: 59 }
]
const sortedArray = someArray.sort((a,b) => {
    return b.rank - a.rank
})
