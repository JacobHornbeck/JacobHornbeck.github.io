Accessing Forms By Their Names

SitePoint JavaScript - Novice to Ninja

In the first part of this chapter is shows a form:

HTML
<form name='search' action='/search'>
    <input name='searchInput'>
    <button type='submit'>Search</button>
</form>

And then is shows it being accessed in the following three ways:

JavaScript
// Only works if there is only one form or this form is always first
const form = document.forms[0]

const form = document.forms.search

const form = document.forms['search']

I knew the first one, but I didn't know that you could access the form elements by a name.
