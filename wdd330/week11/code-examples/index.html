Async and Await

Team Activity

In the Team Activity some of the code contains async functions. I learned about async and await in my Backend Development class, and since it comes up in this weeks Team Activity, I will put a few examples...

JavaScript
async function someFunction() {
    // ...
    return someValue
}

const returnedValue = await someFunction()

The await keyword is required when calling any async function, which are ones like fetch, any function declared with async, and any function that returns a promise.

Any function that has an await request, needs to be declared as an async function.
