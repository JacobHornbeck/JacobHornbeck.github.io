Learn Your Password

Just remember the word/phrase you put in to start with, and then remember these rules:

  • Because this one is of highest priority, all a's will be @, all i's will be !, and all s's will be $.
  • All o's will be changed to 0 and all e's will be turned to 3.
    If you don't have it replace with symbols, then all i's will be 1.
  • Here is a list of all the changes it can make (unless you have the other ones turned on):
    • A will be /-\
    • C will be (
    • D will be |)
    • H will be |-|
    • I will be ][
    • K will be |<
    • L will be |_
    • M will be |\/|
    • N will be |\|
    • O will be ()
    • V will be \/
    • W will be \/\/
    • X will be ><
